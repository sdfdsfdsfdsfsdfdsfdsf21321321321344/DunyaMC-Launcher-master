**Klonlama**

```console
> git clone https://github.com/bynoment/DunyaMC-Launcher-master
> cd DunyaMC-Launcher-master
> npm install
```
---

**Uygulamayı başlat**

```console
> npm start
```
---

**Build**

| Platform    | Command              |
| ----------- | -------------------- |
| Windows  | `npm run build:win`   |
| macOS    | `npm run build:mac`   |
| Linux    | `npm run build:linux` |
